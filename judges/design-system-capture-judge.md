# Judge: Design System Capture

## Pass criteria

PASS if:

- Logo captured.
- Favicon captured.
- Brand colors documented.
- Fonts documented.
- Button styles documented.
- Form styles documented.
- Desktop and mobile Shopify screenshots captured.
- Signup page uses matching or intentionally compatible styling.
- CTA language matches Benton Drones brand tone.

## Fail criteria

FAIL if:

- Backend pages look unrelated to Benton Drones.
- Logo/colors are missing without documented reason.
- Mobile layout is broken.
- Design assets are pulled in a way that exposes secrets.
