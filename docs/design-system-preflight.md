## Recommended production design approach

For the easiest production launch, make `leads.bentondrones.com` visually match the Shopify storefront enough that the transition from Shopify page to owned signup feels intentional.

The first production launch does not need a perfect pixel match. It needs:

- matching logo
- matching colors
- matching button style
- clear trust language
- mobile-friendly form
- obvious relationship to Benton Drones

After launch, refine the design once real user behavior is observed.
