# Goal: Browser QA Coverage

## Objective

Verify the Lead-Ingest user experience through browser automation.

## Required flows

- `/overview` loads
- `/shopify-preview` loads
- `/changelog` loads
- `/roadmap` loads
- `/domain-setup` loads
- `/api-preflight` loads
- `/signup` loads
- `/signup/default` loads
- Shopify preview CTA reaches signup
- valid signup can be submitted
- invalid signup shows validation failure
- admin login page loads
- admin login works with configured local password
- admin dashboard loads after login
- export links are protected when unauthenticated
- export links work when authenticated

## Required outputs

- PASS/FAIL browser QA report
- screenshots or failure notes when applicable
- exact routes tested
