# Goal: Optimized Shopify Design System Capture

## Objective

Capture the Shopify storefront design system and apply it to the owned backend so the transition to `leads.bentondrones.com` feels intentional.

## Required assets

- Logo files, SVG preferred
- Favicon/app icon
- Brand colors
- Typography
- Button styles
- Form styles
- Header/footer screenshots
- Mobile screenshots
- CTA/copy examples
- Trust/safety language

## Required backend styling targets

- `/signup`
- `/signup/<variant>`
- confirmation page
- `/admin-login`
- `/shopify-preview`
- `/overview`
- `/domain-setup`
- `/api-preflight`

## Non-goals

- Pixel-perfect Shopify theme clone before first launch
- Storing Shopify theme secrets in repo
